Markdown 是写作和记录代码笔记的利器，尤其在 GitHub、VS Code、博客里都能用。它的核心理念是**用最少的符号表示排版**，非常轻量。下面总结基本语法：

---

### 1. 标题

用 `#` 表示标题，`#` 越多层级越低：

```markdown
# 一级标题
## 二级标题
### 三级标题
```

---

### 2. 段落 & 换行

- 普通文字就是段落。
- 换行需要在行尾加两个空格或者用 `<br>`：

```markdown
这是第一行  
这是第二行
```

---

### 3. 强调

- **加粗**：`**文字**` 或 `__文字__`
- _斜体_：`*文字*` 或 `_文字_`
- ~~删除线~~：`~~文字~~`

---

### 4. 列表

**无序列表**（用 `-`、`*` 或 `+`）：

```markdown
- 苹果
- 香蕉
- 橙子
```

**有序列表**（用数字加点）：

```markdown
1. 第一项
2. 第二项
3. 第三项
```

---

### 5. 链接 & 图片

**链接**：

```markdown
[GitHub](https://github.com)
```

**图片**（在链接前加 `!`）：

```markdown
![图片描述](图片URL)
```

---

### 6. 代码

**行内代码**：用反引号 `` ` `` 包起来：

```markdown
这是 `git commit` 命令
```

**代码块**：用三个反引号或者缩进四个空格：

```bash
git status
git add .
git commit -m "提交信息"
```

---

### 7. 引用

用 `>`：

```markdown
> 这是一段引用
````

---

### 8. 水平线

用三个 `-`、`*` 或 `_`：

```markdown
---
```

---

### 9. 表格

```Markdown
| 名称 | 年龄 | 城市 |
|------|-----|------|
| 张三 | 18  | 北京 |
| 李四 | 20  | 上海 |
```
